Dynavolt

A nice DynamoDB library for Node.js

Getting started

DynamoDB is a Key-Value oriented database. Amazon offers a hosted version of it in their cloud. They can back up your data and replicate it accross geographic regions so that it's safe. This library tries to reduce the learning curve associated with using DynamoDB by progressively disclosing features only as you need them. 

        
npm install @optoolco/dynavolt

Create a table

The first thing you will want to do is create a table to put some data into. 

        
const Dynavolt = require('@optoolco/dynavolt')
const { err, table } = await Dynavolt.create('artists')

Batch Reads

The first thing you will want to do is create a table to put some data into. 


const { err } = await table.batchRead([

Batch Writes

The first thing you will want to do is create a table to put some data into. 


const { err } = await table.batchWrite([
  { put: { hash: 'foo', range: 'bar', ... } },
